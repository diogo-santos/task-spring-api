# task-spring-api
Rest service for Task domain

## Getting the code on your computer
- [ ] Java 1.8
- [ ] Maven 3.0+
- [ ] Import the project from GitHub: https://github.com/diogo-santos/task-spring-api

Run the app
```
cd task-spring-api
mvn spring-boot:run
```

## Test the App
Check API documentation at http://localhost:8081/swagger-ui.html
